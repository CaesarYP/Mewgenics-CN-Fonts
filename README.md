# 喵喵的结合汉化字体 Mewgenics CN Fonts

#### 介绍
喵喵的结合汉化后可使用的字体

#### 字体来源
* yuyang 苍耳渔阳体：https://tsanger.cn/product/199
* naikai 内海字体：https://www.fonts.net.cn/font-31762349613.html?previewText=%E5%96%B5%E5%96%B5%E7%9A%84%E7%BB%93%E5%90%88
* maoken 猫啃珠圆体：https://www.fonts.net.cn/font-42220289791.html?previewText=%E5%96%B5%E5%96%B5%E7%9A%84%E7%BB%93%E5%90%88
* muyao 沐瑶软笔手写体：https://www.fonts.net.cn/font-35068393713.html?previewText=%E5%96%B5%E5%96%B5%E7%9A%84%E7%BB%93%E5%90%88

#### 使用说明

1.  在本文档“字体来源”处点击链接浏览字体样式
2.  在本仓库中下载他的对应文件，比如`unicodefont_yuyang.swf`就代表苍耳渔阳体
3.  在steam上选择喵喵的结合-⚙-管理-浏览本地文件

![steam浏览本地文件](https://foruda.gitee.com/images/1772263259048010093/0a6120d6_11787408.png "ScreenShot_2026-02-28_152013_884.png")

4.  将swf文件的名字去掉后缀改为`unicodefont.swf`，并放入`swfs`文件夹（若没有该文件夹请创建一个）

![游戏文件夹](https://foruda.gitee.com/images/1772263375906394856/27740ebd_11787408.png "Quicker_20260228_152240.png") 

![存放位置](https://foruda.gitee.com/images/1772263609080111945/3f27d924_11787408.png "Quicker_20260228_152622.png")

5.  在安装了汉化的情况下，在游戏里选择简体中文即可
