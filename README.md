

# mewgenics_cn_fonts

这是 mewgenics 的中文字体集合项目，包含多个 Unicode 字体文件。

## 项目简介

本项目收录了一系列中文 Unicode 字体文件，存储为 SWF 格式。这些字体可应用于 Flash 开发及相关领域。

## 字体文件

项目包含以下字体文件：

- `unicodefont_maoken.swf` - 毛垦字体
- `unicodefont_muyao.swf` - 木曜字体
- `unicodefont_naikai.swf` - 内海字体
- `unicodefont_yuyang.swf` - 育阳字体

## 使用说明

1. 下载所需的字体 SWF 文件
2. 将字体文件导入到您的 Flash 项目中
3. 在需要使用中文的位置设置相应的字体
