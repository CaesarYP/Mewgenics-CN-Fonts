# mewgenics_cn_fonts

This is a collection of Chinese Unicode fonts for mewgenics, containing multiple Unicode font files.

## Project Overview

This project includes a series of Chinese Unicode font files stored in SWF format. These fonts can be used in Flash development and related fields.

## Font Files

The project includes the following font files:

- `unicodefont_maoken.swf` - Mao Ken Font
- `unicodefont_muyao.swf` - Mu Yao Font
- `unicodefont_naikai.swf` - Nai Hai Font
- `unicodefont_yuyang.swf` - Yu Yang Font

## Usage Instructions

1. Download the required font SWF file.
2. Import the font file into your Flash project.
3. Set the corresponding font at locations where Chinese text is needed.
